Como criar um aplicativo para smartphones mais acessível?



Esta página é dedicada para programadores que necessitam de orientações e dicas para a construção de um programa para smartphones acessível.
Abaixo, estão listadas dicas para garantir que várias pessoas conseguirão acessar o seu aplicativo normalmente:

    Para o código ser acessível:

  1. Escolher os sistemas operacionais que o seu programa irá rodar
  2. Escolher a linguagem de programação adequada
  3. Identar o código adequadamente
  4. Acrescentar comentários sempre que possível


    Para o executável ser acessível:

  • Menus intuitivos com uso de imagens, sons simultâneos e recursos tátil (vibração) ;
  • Vocabulário simples;
  • Recursos visuais amplificados;
  • Fornecer um equivalente de texto e áudio para qualquer imagem importante;
  • Representação visual das informações sonoras apresentadas em texto e/ou LIBRAS;
  • Os sons devem ser claros e sem ruídos;
  • Possibilidades iguais de controle com comandos de voz, teclado e/ou touch;
